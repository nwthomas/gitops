Thank you for installing ArgoCD!

Your release is named {{ .Release.Name }}.

To access the ArgoCD server, you can:

1. Port forward to the ArgoCD server:
   kubectl port-forward svc/{{ .Release.Name }}-server -n {{ .Release.Namespace }} 8080:443

2. Or access via LoadBalancer (if configured):
   kubectl get svc {{ .Release.Name }}-server -n {{ .Release.Namespace }}

3. Or access via Ingress (if configured):
   kubectl get ingress -n {{ .Release.Namespace }}

The default admin password can be retrieved with:
kubectl -n {{ .Release.Namespace }} get secret {{ .Release.Name }}-server -o jsonpath="{.data.admin\.password}" | base64 -d

For more information on ArgoCD, visit:
https://argo-cd.readthedocs.io/
